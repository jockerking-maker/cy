# 手把手教你封装属于自己的 Windows ISO 镜像，让重装变得超简单

> **来源**: [知乎 - 红头发蓝胖子](https://zhuanlan.zhihu.com/p/4130861066)  
> **发布时间**: 2024-10-30

## 引言

嗨，大家好，我是红头发蓝胖子。

每次重装系统最大的烦恼就是重新安装软件，复制资料，这往往要折腾一天甚至更久的时间。

如果我们把常用的软件、系统优化，必备的资料，都统一封装到一个 ISO 镜像，重装系统后，所有需要的东西，都已经内置了，岂不是完美。

本期内容，我将分享如何封装一个属于你自己的 Windows ISO 镜像。

---

## 需要准备的工具

1. Windows ISO 镜像
2. VMware 虚拟机
3. AnyBurn（光盘刻录和镜像文件管理工具）

> 所有的工具都是免费的！

---

## 第一步：下载 Windows ISO 镜像

Windows 的安装镜像可以直接从微软官网下载，这里下载 Windows 11 ISO 镜像。

- 选择下载 → 选择 **Windows 11 适用于 x64 设备的多版本 ISO**
- 这个版本里包含家庭版和专业版等，安装的时候可以自由选择
- 点击立即下载，系统会验证一下请求
- 接着选择系统的语言，这里选择**简体中文**
- 然后会出现一个 64-bit download 的链接
- 点击开始下载，也可以复制链接到下载工具中下载

![下载Windows ISO](图片占位：微软官网下载Windows11 ISO界面)

---

## 第二步：下载 VMware 17 Pro

> 更多教程内容，可以参考作者的另一期教程

### 创建虚拟机

- 点击 **创建虚拟机**，默认选择**典型**，点击下一步
- 安装程序光盘映像文件 ISO，这里选择下载好的 Windows 11 23H2 的镜像
- VMware 已经检测到 Windows 11 x64 的系统，点击下一步
- 设置虚拟机的名称和位置，点击下一步

### TPM 加密设置

由于 Windows 11 需要可信模块才能运行，所以这里需要设置一个密码：

- 加密类型保持默认，选择 **"只有支持 TPM 所需的文件才需要加密"**
- 点击**生成**，可以自动生成密码，然后点击复制
- **⚠️ 这个密码需要备份出来保存一下**，以后有虚拟机的调整可能会用到
- 点击下一步

![VMware创建虚拟机](图片占位：VMware创建虚拟机配置界面)

### 磁盘和硬件配置

- 设置虚拟机的磁盘容量
- 选择**将虚拟磁盘存储为单个文件**，点击下一步
- 点击**自定义硬件**，可以调整一下内存和 CPU 的大小，让虚拟机的性能更高一点
- 然后点击**完成**

![自定义硬件](图片占位：VMware自定义硬件设置界面)

---

## 第三步：安装 Windows 11 系统

- 开启虚拟机，开始安装 Windows 11 系统
- 将光标切入到虚拟机，**按任意键**，从 Windows ISO 映像启动
- 点击下一步
- 可以点击**没有产品密钥**，跳过激活
- 选择 Windows 11 的版本：**Windows 11 专业版**
- 点击下一步
- 许可协议，点击**接受**
- 安装类型：由于是全新的安装，选择**自定义：仅安装 Windows**
- 选择 Windows 11 安装的磁盘位置，这里仅有一个磁盘，点击下一步即可

进入安装过程，稍等一会。期间会有数次重启。接着进入到 Windows 的设置向导，设置地区、语言、键盘、设备名称、用户名等。

设置完成后，稍等一会，就可以完成。

> ✅ **Windows 11 安装完成！**

![Windows安装过程](图片占位：Windows11安装过程截图)

---

## 第四步：安装 VMware Tools

接着为了后续更好的操作虚拟机，安装 VMware Tools：

- 在 VMware 菜单中，点击**虚拟机**
- 选择**安装 VMware Tools**
- VMware 会在系统中挂载一个 DVD 驱动器
- 打开 DVD 驱动器（D 盘）
- 找到 `VMwareTools`
- 点击 `setup64`
- 默认选择**典型安装**

安装好 VMware Tools 就可以调整虚拟机的屏幕分辨率等。

- 安装完成后**重启**

重启后，一个新的 Windows 11 系统就已经完成。

---

## 第五步：增加虚拟机硬盘

- **关闭** Windows 11 虚拟机
- 点击**编辑虚拟机设置**
- 点击**添加** → 选择**硬盘** → 点击下一步
- 选择 **NVMe** → 下一步
- **创建新的虚拟机磁盘** → 点击下一步
- 大小默认 64GB 就够用了
- 选择**将虚拟机硬盘存储为单个文件**
- 命名磁盘文件，保持默认即可 → 点击**完成**

看到这样虚拟机上有 2 个磁盘。为了在系统中更好的区分，可以将 2 个硬盘的大小设置为不一样。

![添加硬盘](图片占位：VMware添加第二块硬盘界面)

### 初始化新硬盘

开启虚拟机，新的硬盘需要设置一下才可以使用：

- 右键点击**此电脑** → 显示更多选项 → 点击**管理**
- 点击**磁盘管理**，会提示初始化磁盘
- 设置磁盘分区的形式，这里默认选择 **GPT 的分区表**，点击确定
- 右键点击磁盘 1，选择**新建简单卷**
- 点击下一页，都保持默认即可

磁盘创建成功后就可以在**此电脑**中看到 E 盘。

![磁盘管理](图片占位：Windows磁盘管理初始化界面)

## 第六步：手动优化系统

这时候进入虚拟机的操作系统：

- 调整隐私和安全性设置，将一些设置直接关闭掉
- 也可以在设置中，根据自己的实际使用需求进行优化，关闭一些不必要的功能

## 第七步：使用 Optimizer 优化

也可以使用软件 **Optimizer** 对 Windows 进行更深度的优化。

使用软件，更适合普通用户，更快速的关闭和打开一些不必要的服务和启动项等等。

![Optimizer](图片占位：Optimizer软件优化界面)

接着就可以安装所需要的软件，比如播放器等等，常用的软件都可以安装到系统中。

## 第九步：创建 install.wim 文件（关键步骤）

### 挂载 ISO 并进入恢复环境

- 编辑虚拟机设置，在 **CD/DVD 挂载映像**，选择 ISO 映像文件（Windows 11 的安装文件）
- 然后点击确认，**开启虚拟机**
- 按 **F2** 进入 BIOS，调整启动顺序，从 **CDROM 启动**
- 然后按任意键进入到 Windows 11 安装过程
- 点击下一步

> ⚠️ **这里不是进行真正的安装**，需要在这个过程中执行一些操作！

- 点击**修复计算机**
- 点击**疑难解答**
- 点击**命令提示符**

![修复计算机](图片占位：Windows安装界面-修复计算机)

### 执行 DISM 命令

进入命令提示符后，依次操作：

```cmd
C:
```
进入到 C 盘的目录，输入 `dir`，查看目录中的文件夹，包含 `Windows` 为系统盘。

```cmd
D:
```
进入到 D 盘，输入 `dir` 查看目录中文件，该磁盘是新创建的所以为空。

然后输入关键命令：

```cmd
dism /capture-image /imagefile:D:\install.wim /capturedir:C:\ /Name:Windows11_Custom
```

> ⚠️ **注意**：这里要注意斜杠的方向和空格，不要打错！

启动部署映像服务和管理工具，开始制作映像，会有一个进度条的提示，大概 **10-20 分钟**，可以完成操作。

![DISM制作映像](图片占位：DISM命令执行进度条界面)

### 复制 install.wim 文件

- 关闭命令提示符窗口，点击继续，进入到 Windows 11
- 打开 **E 盘**，可以看到刚才制作的 `install.wim` 文件

> 💡 这里解释一下为什么刚才制作在 D 盘，但是这里是 E 盘：因为挂载的 ISO 镜像占用了 D 盘的盘符，所以之前在命令提示符中的 D 盘就显示到了 E 盘。

- 右键点击文件 → 选择**复制**
- 回到宿主机上 → 点击**粘贴**

将 `install.wim` 文件从虚拟机复制出来。

## 第十步：AnyBurn 工具介绍

接着用到的工具 **AnyBurn**，一个完全免费的光盘刻录和镜像文件管理工具。

- 可以直接从官网下载
- 支持中文
- 个人版完全免费，专业版免费试用
- 专业版仅比个人版多了转换音乐文件格式和安装 Windows 到 U 盘
- 我们下载**个人版**足够满足需求

![AnyBurn](图片占位：AnyBurn软件主界面)

---

## 第十一步：封装 ISO 镜像

安装非常简单，一路下一步即可。打开 AnyBurn：

- 有非常多的功能，这里我们点击**编辑镜像文件**
- 选择镜像文件，这里选择 Windows 11 的**官网原版 ISO 镜像**
- 然后点击下一页，可以看到镜像中的文件

![编辑镜像文件](图片占位：AnyBurn选择镜像文件界面)

- **双击**进入到 `source` 文件夹
- 找到 `install.wim` 文件
- 点击菜单中的**删除**
- 然后点击**增加**
- 将我们之前制作的文件，添加进来

![替换install.wim](图片占位：AnyBurn替换install.wim文件界面)

- 点击下一页
- 设置新镜像的文件名称和保存路径，这里改成 `win11-custom`

这时候就开始封装我们自己的镜像文件了。稍等一会即可制作完成，生成镜像文件成功。

> ✅ **现在你就有了一个自己定制的 ISO 镜像文件！**

---

## 第十二步：验证封装的 ISO 镜像

ISO 镜像文件制作完成后，我们来验证一下是否可以正常安装系统：

- 快速地新建一台虚拟机
- 在引用 ISO 文件的时候，导入我们刚才自定义的 ISO 文件 `win11_custom`

安装完成后和之前自定义调试的系统内容是完全一致的。
