# CodeBuddy vs WorkBuddy 对比

## 核心区别概览

| 维度 | CodeBuddy（AI 编程助手） | WorkBuddy（办公助手） |
|:--|:--|:--|
| **定位** | AI **编程助手**，嵌入 IDE（VS Code / JetBrains） | AI **办公助手**，嵌入企业微信/协作场景 |
| **核心场景** | 代码编写、调试、项目开发、架构分析 | 会议纪要、日程管理、审批流转、文档协作 |
| **用户群** | 开发者 / 技术团队 | 企业全体员工 |
| **交互方式** | IDE 内对话、自动补全、代码审查 | 聊天对话、快捷指令、流程机器人 |
| **深度能力** | 理解完整项目结构、重构、单元测试、CI/CD | 审批流、周报生成、公告推送、组织通讯录 |

---

## CodeBuddy 能做什么

### 1. 代码开发

| 能力 | 说明 |
|:--|:--|
| **写代码** | 根据需求生成完整功能模块（前端 / 后端 / 脚本） |
| **改代码** | 精确替换、重构、优化代码 |
| **查代码** | 全局搜索符号、引用、调用链，快速定位问题 |
| **Debug** | 分析错误栈、修 lint 错误、优化性能 |

### 2. 多文件 / 复杂任务

| 能力 | 说明 |
|:--|:--|
| **任务清单** | 自动创建任务清单，分步执行大型需求 |
| **批量处理** | 同时处理多个文件，批量化改动 |
| **Team Mode** | 创建多个 AI 子代理并行工作（如一个写前端、一个写后端） |

### 3. 联网搜索 & Web

| 能力 | 说明 |
|:--|:--|
| **实时搜索** | 搜索技术文档、最新 API |
| **网页抓取** | 抓取网页内容进行分析 |
| **浏览器自动化** | 操作网页、截图、填表单 |

### 4. 文档生成（需加载技能）

| 能力 | 说明 |
|:--|:--|
| **PPT** | 从零创建专业幻灯片 |
| **PDF** | 读取、合并、拆分、加水印、OCR |
| **Word** | 生成带目录 / 页眉 / 排版的报告 |
| **Excel** | 处理表格、公式、图表、数据清洗 |

### 5. 多模态生成

| 能力 | 说明 |
|:--|:--|
| **文生图** | 根据描述生成图片 |
| **文生视频 / 图生视频** | AI 生成视频内容 |
| **图片特效** | 动效处理 |

### 6. 自动化 / 定时任务

| 能力 | 说明 |
|:--|:--|
| **定时任务** | 设置每日 / 每周定时任务（如每天早上拉代码检查、定时发日报） |
| **一次性提醒** | 指定时间触发提醒 |

### 7. 部署 & 数据库

| 能力 | 说明 |
|:--|:--|
| **部署** | CloudBase / EdgeOne Pages / Lighthouse 一键部署项目 |
| **数据库** | Supabase / CloudBase 连接云数据库、读写数据 |

### 8. 知识库

| 能力 | 说明 |
|:--|:--|
| **RAG 问答** | 连接项目文档 / 私有知识库，基于知识库精准回答 |

---

## 一句话总结

> **WorkBuddy** 帮你管工作和办公流程，**CodeBuddy** 帮你看代码、写代码、修代码、部署代码，是整个开发周期里的 AI 搭档。
