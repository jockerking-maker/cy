# 《西游记》大纲

---

## 一、作品概览

- **作者**：吴承恩（明代）
- **体裁**：章回体神魔小说
- **核心主线**：唐僧师徒四人西行取经，历经九九八十一难，终得真经
- **主题**：修心证道、反抗权威、团队成长、佛道融合

---

## 二、主要人物群像

| 人物 | 身份 | 性格关键词 | 核心弧光 |
|------|------|-------------|----------|
| **唐僧**（玄奘） | 金蝉子转世、取经人 | 慈悲、固执、脆弱 | 从怯懦到坚定 |
| **孙悟空** | 天生石猴、齐天大圣 | 桀骜、机智、忠诚 | 从野性到自律 |
| **猪八戒**（猪悟能） | 天蓬元帅转世 | 贪吃、好色、懒散但可爱 | 从散漫到担当 |
| **沙僧**（沙悟净） | 卷帘大将贬下凡 | 忠厚、沉默、勤恳 | 稳定器般的存在 |
| **白龙马** | 西海龙王三太子 | 隐忍、忠诚 | 赎罪与奉献 |

---

## 三、总体结构（三大部分）

### 第一部分：大闹天宫（第 1—7 回）
- 石猴出世，花果山称王
- 拜师菩提祖师，学得七十二变、筋斗云
- 龙宫夺宝（如意金箍棒）、地府销名
- 官封弼马温→齐天大圣
- 大闹天宫，被压五行山

### 第二部分：取经缘起（第 8—12 回）
- 如来传经，观音寻取经人
- 玄奘身世与出家
- 唐太宗游地府，水陆大会
- 玄奘领旨西行

### 第三部分：西行取经（第 13—100 回）

**前期收徒（第 13—22 回）**
- 五行山收悟空
- 鹰愁涧收白龙马
- 高老庄收八戒
- 流沙河收沙僧

**中期历难（第 23—80 回）**
- **三打白骨精**（第 27 回）—— 悟空被逐
- **宝象国救师**（第 28—31 回）—— 悟空回归
- **平顶山**金银角大王（第 32—35 回）
- **乌鸡国**除妖（第 36—40 回）
- **红孩儿**火云洞（第 40—42 回）
- **车迟国**斗法（第 44—46 回）
- **通天河**灵感大王（第 47—49 回）
- **真假美猴王**（第 56—58 回）
- **火焰山**三借芭蕉扇（第 59—61 回）
- **祭赛国**除九头虫（第 62—63 回）
- **小雷音寺**黄眉怪（第 65—66 回）
- **盘丝洞**蜘蛛精与黄花观（第 72—73 回）
- **狮驼岭**三魔（第 74—77 回）
- **比丘国**救小儿（第 78—79 回）

**后期收官（第 81—100 回）**
- **灭法国**剃度（第 84—85 回）
- **天竺国**玉兔精（第 93—95 回）
- **寇员外**还魂（第 96—97 回）
- **凌云渡**脱胎（第 98 回）
- 灵山见如来，取回真经
- 通天河老鼋翻船，最后一难
- 五圣成真，全书终

## 四、核心主题与深层寓意

| 主题 | 体现 |
|------|------|
| **修心证道** | 每一难都是心魔外化，悟空代表"心猿"的驯服 |
| **反抗与秩序** | 大闹天宫是个人反抗的巅峰，取经则是融入秩序的旅程 |
| **团队互补** | 师徒四人性格迥异、缺陷互补，构成取经团队 |
| **讽刺与幽默** | 大量对官场、宗教、人性的暗讽藏在奇遇背后 |
| **命数与自由** | 看似一切劫难皆注定，实则仍需个人主动选择与行动 |

## 五、经典回目路线图（简版）

```
大唐 → 两界山 → 蛇盘山 → 高老庄 → 流沙河
     → 五庄观 → 白虎岭 → 宝象国 → 平顶山 → 乌鸡国
     → 火云洞 → 车迟国 → 通天河 → 女儿国 → 火焰山
     → 祭赛国 → 小雷音 → 朱紫国 → 盘丝洞 → 狮驼岭
     → 比丘国 → 无底洞 → 灭法国 → 天竺国 → 灵山
```

## 六、阅读建议

- **精读回目**：第 1—7 回（大闹天宫）、第 27 回（三打白骨精）、第 59—61 回（三借芭蕉扇）、第 98—100 回（取经结局）
- **跳读策略**：部分降妖故事模式类似（遇妖→悟空打不过→请神仙帮忙），初期可先看各难开头与结尾
- **延伸角度**：可结合《大唐西域记》（玄奘真实游记）对比虚实
